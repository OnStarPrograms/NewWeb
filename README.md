# New-Web website  

## Overview  

This was my 3rd website I made (circa: 2024). I created this website to learn how to build a dynamic client side website generator using the DOM and J.S.. This websites engine pulls a pre-made JSON file that holds all the information for this website from a webserver I'm currently running with my [current website](https://starprograms.dev) And dynamically build a multipage _(simulated)_ website on a single _(actual)_ webpage.  

## Why Is This Still Online?  

I believe that if I keep my previous websites online. Then whoever looks at what I can do now, can see the hard journey I went through navigating thes older programming methods. This is also so I can look to my past and see how far I've gone aswell.
